# n8n-payment-automation-system
An enterprise-grade automation workflow designed to streamline accounts receivable. This system monitors financial records in Google Sheets, identifies overdue payments, and uses a Python-based aggregation engine to send consolidated, HTML-formatted email reports to clients, reducing manual overhead and email fatigue.
